Data Storage Locations

Student Names

Stored in: students_data.json
Edit this file directly to add/remove students. Click "Reload Students" to refresh.
Format: { "classes": { "ClassName": [{"id": "...", "name": "..."}] } }

Scores & Grades

Stored in: localStorage["curve_manager_data_v1"]
Contains MCQ scores, FRQ scores, and all calculated values per class.
Persists automatically on every change.

Student Order

Stored in: localStorage["curve_manager_order_v1"]
Drag-and-drop reordering is saved immediately. Order persists between sessions.

AP Curve Bands

Stored in: localStorage["curve_manager_ap_bands_v1"]
Edit the raw/curve min/max values in the legend panel. Changes save automatically.

Formula Settings

Stored in: localStorage["curve_manager_formula_v1"]
The curve formula is saved per class. Edit in the Formula tab.

Assignment Snapshots

Stored in: localStorage["curve_manager_assignments_v1"]
Saved assignments with all scores at that point in time. Export to CSV.

Reset to Defaults

Click "Reset All" to clear all localStorage data and reload students from students_data.json.
This does NOT modify the JSON file.

N/A Filtering

Empty strings, null, undefined, and "N/A" values are automatically filtered out.
Only actual student data with valid names is displayed.
